# AI 연구팀 운영

이 폴더는 “AI가 스스로 연구팀처럼 움직이기” 위한 역할/프로토콜을 정의합니다.

## 역할

- `lead.md`: 연구 리드(스코프/우선순위/의사결정)
- `scout.md`: 문헌 스카우트(검색/선별/요약)
- `experimenter.md`: 실험가(프로토타입/평가/재현성)
- `writer.md`: 작가(논리 전개/도표/원고)
- `reviewer.md`: 리뷰어(반증/약점/관련연구 누락 점검)

## 운영 규칙(간단)

- 모든 토픽은 `topics/<id>-<slug>/` 아래에서만 작업
- 모든 주장(claim)은 “근거(논문/실험/데이터)”에 연결
- 결정은 `topics/.../decisions.md`에 기록
