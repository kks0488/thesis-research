# Roadmap

목표: “미래먹거리” 관련 최신 연구(Thesis/논문)를 지속적으로 수집·정리하고, 그 위에서 **새로운 연구 가설/프로토타입**을 만들어 **논문 형태로 발표**한다.

## 마일스톤

1. 문헌 파이프라인 고정
   - 검색 쿼리/출처/선정 기준 문서화
   - 메타데이터 저장 형식(JSONL/CSV) 일원화
2. 토픽 1개를 end-to-end로 완주
   - 문제정의 → 관련연구 → 가설 → 실험 설계 → 결과 → 논문화
3. 토픽 3개로 확장
   - 공통 실험/평가지표/데이터셋 재사용
4. 발표 패키지
   - 논문 PDF + 슬라이드 + 데모(가능하면)

## 선정 기준(초안)

- “미래먹거리” 신호: 지속가능성, 비용/스케일, 영양/안전, 규제/공급망, 소비자 수용성
- 논문 유형: thesis(학위논문) 우선 + 관련 peer-reviewed + 최신 preprint
- 실용성: 실험 가능(소규모 프로토타입/모의실험/데이터 기반 분석 가능)
